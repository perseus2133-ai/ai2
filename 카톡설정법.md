# 카카오톡 알림 설정법 (1회, 약 10분)

매일 새벽 크롤이 끝나면 **AI 데일리 3선이 카톡 "나와의 채팅"으로** 도착합니다.
카카오 정책상 계정 인증은 본인만 할 수 있어서 아래 1회 설정이 필요합니다.

## 1단계 — 카카오 개발자 앱 만들기

1. https://developers.kakao.com 접속 → 카카오 계정 로그인
2. 상단 [내 애플리케이션] → **[애플리케이션 추가하기]**
   - 앱 이름: `ai2 알림` (아무거나) / 회사명: 개인 이름
3. 만든 앱 클릭 → 좌측 **[앱 키]** → **REST API 키** 복사해두기

## 2단계 — 카카오 로그인 설정

1. 좌측 **[카카오 로그인]** → 활성화 설정 **ON**
2. 같은 페이지의 **Redirect URI** → [URI 추가] → 정확히 입력:
   ```
   https://localhost
   ```
3. 좌측 **[카카오 로그인] > [동의항목]** →
   **"카카오톡 메시지 전송 (talk_message)"** 찾아서 → **선택 동의** 설정

## 3단계 — 토큰 발급 (PC에서 1회 실행)

ai2_repo 폴더에서:

```bash
python kakao_auth_setup.py
```

1. REST API 키 붙여넣기 → 브라우저가 열림 → 카카오 로그인 + 동의
2. 주소창이 `https://localhost/?code=XXXX` 로 바뀜 (페이지는 "연결할 수 없음" 에러여도 정상!)
3. 주소창의 `code=` 뒤 값을 복사해 터미널에 붙여넣기
4. 성공하면 **KAKAO_REST_KEY / KAKAO_REFRESH_TOKEN 두 값이 출력**되고,
   카톡 "나와의 채팅"에 테스트 메시지가 도착합니다 📨

## 4단계 — GitHub Secrets 등록

1. https://github.com/perseus2133-ai/ai2/settings/secrets/actions 접속
2. **[New repository secret]** 으로 2개 등록:

| Name | Value |
|---|---|
| `KAKAO_REST_KEY` | 3단계에서 출력된 값 |
| `KAKAO_REFRESH_TOKEN` | 3단계에서 출력된 값 |

3. (선택) `APP_URL` — 카톡 메시지 버튼이 열 주소. Streamlit 앱 주소를 넣으면
   버튼 누를 때 바로 스크리너가 열립니다. 미설정 시 GitHub 저장소로 연결.

끝! **다음 새벽 크롤부터 매일 카톡이 옵니다.**

## 참고 — 토큰 수명

- 리프레시 토큰은 **2개월 유효**, 매일 사용하면 만료 전 카카오가 새 토큰을
  발급해줍니다. 새 토큰이 발급되면 Actions 로그에 안내가 출력되는데:
  - (선택) `GH_PAT` 시크릿(repo secrets 쓰기 권한 Personal Access Token)을
    등록해두면 **자동으로 교체**됩니다.
  - 없으면 두 달에 한 번쯤 로그의 새 토큰을 Secrets에 수동 교체하면 됩니다.
- 카톡이 안 오기 시작하면: Actions 로그의 "📨 카카오톡 알림" 스텝 확인 →
  대부분 토큰 만료이니 3단계만 다시 실행하면 됩니다.
